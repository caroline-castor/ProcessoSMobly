Teste para Desenvolvedor PHP - Mobly
Desenvolvido por Caroline Castor dos Santos 

Para o desenvolvimento foi utilizado o servidor APACHE executado pelo XAMPP for Windows 7.0.23

REQUERIMENTOS:
	Servidor Apache 2.4 (XAMPP for Windows 7.0.23)
	PHP 7.0.23 (incluso no XAMPP)
	MYSQL 5.7

EXECUÇÃO:
	Executar o script de criação do banco de dados disponível em /posts/scriptBD.sql
	Colar o diretório /posts em htdocs
	Alterar, caso necessário, a login/senha do MYSQL em /posts/ConectaBD.php (Default:  user:root, pass:root)
	Iniciar Server Apache
	Acessar localhost/posts
